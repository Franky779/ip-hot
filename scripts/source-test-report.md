## 四、核心修复记录

本次测试中修复的关键问题：

1. **LLM prompt评分过严** — 游戏资讯被误判为"纯游戏资讯"全部过滤，已加明确说明
2. **CDP端口配置错误** — 脚本配了9222（Chrome原生端口），应走3456（Proxy）
3. **CDP eval引号冲突** — 选择器含双引号破坏JS语法，改用JSON.stringify转义
4. **Supabase upsert** — 缺少`on_conflict=source,url`和`return=representation`
5. **入库去重** — 同batch内重复URL导致Supabase 500错误，已加URL去重
