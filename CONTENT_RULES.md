# IP-hot 资讯上架标准与约束

> 本文件用于定义哪些资讯可以上架展示、哪些需要过滤，供随时更新优化。

---

## 一、必须过滤的内容（不上架）

### 1.1 纯娱乐/八卦类
- 明星私生活、绯闻、恋情
- 无关 IP/ACG/文创行业的娱乐新闻

### 1.2 无行业价值的内容
- 单纯的电影/剧集观后感、书评
- 个人博客随笔
- 招聘信息

### 1.3 低质量/重复内容
- 标题党、无实质内容
- 与已上架资讯高度重复
- 仅有图片/视频无文字说明

### 1.4 特定排除关键词
| 关键词模式 | 说明 |
|-----------|------|
| `to star in` / `cast as` | 纯演员选角新闻 |
| `review` / `retrospective` | 影评/书评 |
| `joins the cast` | 演员加盟 |

## 二、优先收录的内容（高价值）

### 2.1 IP 产业核心
- [x] IP 授权、联名合作案例
- [x] 动漫/游戏新作发布、PV 公开
- [x] 潮玩/谷子/盲盒新品
- [x] 文创产品商业化动态

### 2.2 产业趋势
- [x] 行业投融资、并购
- [x] 政策变化、法规解读
- [x] 市场规模、消费数据
- [x] 出海/国际化动态

### 2.3 展会活动
- [x] 漫展、潮玩展、授权展信息
- [x] 品牌快闪店、主题展
- [x] 行业峰会、论坛

### 2.4 技术/创新
- [x] AI + 内容创作
- [x] 虚拟偶像、数字藏品
- [x] 新技术在 IP 领域的应用

---

## 三、内容质量要求

| 维度 | 标准 |
|------|------|
| **时效性** | 优先 7 天内资讯，最长不超过 30 天 |
| **原创性** | 优先一手消息，二手编译需有价值增量 |
| **准确性** | 来源可靠，数据有出处 |
| **相关性** | 与动漫/IP/ACG/文创/潮玩/非遗/文博直接相关 |

---

## 四、分类标准

上架资讯必须归入以下 5 个分类之一：

1. **新作发布** — 新作品、新角色、新 PV、新品上市
2. **IP授权** — 联名合作、授权案例、品牌合作
3. **潮玩谷子** — 盲盒、手办、周边、谷子经济
4. **产业动态** — 投融资、政策、数据、公司动态
5. **展会活动** — 展览、峰会、快闪、发布会

---

## 五、推荐理由要求

每条上架资讯应附 1-2 句点评，要求：
- 有行业洞察，不泛泛而谈
- 点出对 IP 从业者的启发
- 可指出风险或机会
